# Assignment 2 - Web Design Portfolio

## Description
O Assignment 2 consiste num projeto de webdesign de um portfolio interativo, com animações em CSS e JavaScript. O objetivo é criar uma apresentação visual dinâmica e moderna dos conteúdos do portfolio.

O website integra uma API que muda a key de hora em hora, mas está sempre a buscar os dados corretamente. Para atualizar a key, basta ir à página da API e obter uma nova: [Spotify Developer](https://developer.spotify.com/).

O site utiliza também uma biblioteca apenas para carregar uma fonte do Adobe Fonts. Esta biblioteca serve apenas para usar a fonte e não implementa nenhuma funcionalidade adicional.

## Features
- Layout de portfolio responsivo
- Animações suaves com CSS e JavaScript
- Integração com API externa (Spotify)
- Fonte personalizada via Adobe Fonts

## Visuals
Inclui elementos animados no design do portfolio para melhorar a experiência do utilizador.
